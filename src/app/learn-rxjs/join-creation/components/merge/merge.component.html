Merge Operator

Combines a number of observables stream and concurrently emits all values from every given input stream. As values from any combined sequence are produced, those values are emitted as part of the resulting sequence. Such process is often reflected to as flattering in doc.

Use Case: Use this operator if you are not concerned with the order of emission and is simply interrested in all values coming out from multiple combined streams as they are produced by one stream.
