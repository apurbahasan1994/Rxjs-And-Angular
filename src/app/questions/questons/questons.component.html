1. What is Dependency Injetion in Angular?

Ans: Dependency Injection is a coding pattern in which a class asks for dependencies from external source rather than creating them itself.

2. What is router state?

Ans: Router state is a tree of activated routes. Every node in this tree knows about the 'consumed' url segments, the extracted parameters and the resolved data. We can access the current RouterState from anywhere in the applicatiob using the router service and the routerstate property

3. Directive?

Ans: Functions that are executed when they are found on the dom by the angular compiler

Examples: Component, Attribute , Structural, Custom
